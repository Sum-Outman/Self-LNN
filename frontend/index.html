 API密钥使用说明

                                    

                                        
所有API调用需要在HTTP头中携带 X-API-Key 字段进行认证。

                                        

                                            cURL 示例:
                                            
curl -H "X-API-Key: your_key_here" http://localhost:8080/api/status

                                        

                                        

                                            Python 示例:
                                            
import requests
response = requests.get(
    'http://localhost:8080/api/status',
    headers={'X-API-Key': 'your_key_here'}
)

                                        

                                        

                                            JavaScript 示例:
                                            
fetch('http://localhost:8080/api/status', {
    headers: { 'X-API-Key': 'your_key_here' }
})
