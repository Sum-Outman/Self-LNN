API 使用示例 / Examples

GET/api/system/diagnostic

cURL

curl -H "X-API-Key: YOUR_KEY" http://localhost:8080/api/system/diagnostic

Python

import requests
r = requests.get("http://localhost:8080/api/system/diagnostic",
    headers={"X-API-Key": "YOUR_KEY"})
print(r.json())

JavaScript (fetch)

fetch("http://localhost:8080/api/system/diagnostic", {
    headers: { "X-API-Key": "YOUR_KEY" }
}).then(r => r.json()).then(d => console.log(d));
POST/api/multimodal/learn
curl -X POST http://localhost:8080/api/multimodal/learn \
  -H "Content-Type: application/json" \
  -H "X-API-Key: YOUR_KEY" \
  -d '{"fusion":"concat","mode":"joint","modalities":"vision,audio,text"}'
GET/api/capability/diagnose
curl -H "X-API-Key: YOUR_KEY" http://localhost:8080/api/capability/diagnose

返回12项AGI能力开关状态：自我认知/决策/执行/学习/演化/模仿/修正/反思/好奇心/规划/对话/并发

GET/api/devices/discover
curl -H "X-API-Key: YOUR_KEY" http://localhost:8080/api/devices/discover

扫描摄像头/音频/串口/ROS/Gazebo等所有可用设备
